Installing
----------

Install nodejs and then run:

```sh
# install the cli
$ npm install -g htmllint-cli
# create a .htmllintrc file
$ htmllint init
$ htmllint [files...]
# an example:
$ htmllint public/*.html
$ htmllint --help # for more info
```
